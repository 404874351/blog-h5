# blog-h5

### 介绍

个人博客-客户端-h5

本项目实现了基础博客系统的h5版本，仅做了一些样式上的修改，具体内容参见 https://github.com/404874351/blog-client 。

### 项目使用

```bash
# 克隆项目到本地
git clone
# 下载依赖
npm install
# 修改/src/config/index.js中的域名
domain = <your-domain>
qiniu_domain  = <your-qiniu-domain>
# 启动服务
npm run serve
# 启动测试
npm run test
# 项目打包
npm run build
```
